## （一）结构体的定义和初始化
#### 1.结构体的定义
以创建一个学生的结构体为例，来了解结构体的定义格式
```c
//学生
struct Stu
{
	//成员变量
	char name[20];
	int age;
	char sex[10];
	char tele[12]
};
```
* 这样相当于就创建了一个自定义变量类型struct Stu
* 那么就可以创建这个类型的变量，eg：struct Stu s;
* s就是一个结构体变量
		
 
#### 结构体变量的初始化
```c
struct Stu s = { "zhanfsan",20,"nan","15555555665" };
```
